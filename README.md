# zshconfig

## Installation
1. Install Oh My Zsh:
```sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"```

2. Setup:

    - `chmod +x scripts/*`
    - `./scripts/setup.sh`

3. Work Configs

    To keep work specific configuration seperate from your global config:

    - `./scripts/new_workconfig.sh {name}`
