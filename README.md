# zshconfig

## Installation
1. Install Oh My Zsh:
```sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"```

2. At top of `zshrc.sh` update `ZSH_CONFIG_DIR` with path to project directory

3. Copy contents to zshrc:
```cp zshrc.sh ~/.zshrc```

4. Execute new zshrc:
```source ~/.zshrc```
